Detailed Description

template<typename GV, class MI>
class Dune::Functions::RannacherTurekPreBasis< GV, MI >

Pre-basis for a Rannacher-Turek basis.

These are Crouzeix-Raviart-elements for quadrilateral elements. See Rolf Rannacher and Stefan Turek. Simple nonconforming quadrilateral Stokes element. Numerical Methods for Partial Differential Equations, 8:97â€“111, 1992.
