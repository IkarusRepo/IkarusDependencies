Detailed Description

Lexicographic merging of direct children without blocking.

Example: For two children {f} and {g} with multi-indices (all i*,k* can be multi-indices, themselves)

function in {f} index
f_0 (0,i0)
f_1 (0,i1)
f_2 (1,i2)
function in {g} index
g_0 (0,k0)
g_1 (1,k1)
g_2 (2,k2)

the merged indices will be

function in {f,g} index
f_0 (0,i0)
f_1 (0,i1)
f_2 (1,i2)
g_0 (2,k0)
g_1 (3,k1)
g_2 (4,k2)
