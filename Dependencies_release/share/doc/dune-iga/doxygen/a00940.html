Detailed Description

template<typename GV, class MI, Dune::IGA::LinearAlgebra NurbsGridLinearAlgebraTraits>
class Dune::Functions::NurbsPreBasis< GV, MI, NurbsGridLinearAlgebraTraits >

Pre-basis for B-spline basis.

Template Parameters
GVThe GridView that the space is defined on
MIType to be used for multi-indices

The BSplinePreBasis can be used to embed a BSplineBasis in a larger basis for the construction of product spaces.
