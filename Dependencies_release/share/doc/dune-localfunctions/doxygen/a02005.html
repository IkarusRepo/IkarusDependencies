Determine coefficients interpolating a given function.

Parameters
ffAn object supporting the expression ff.evaluate(x,y), where x is of type Traits::DomainLocal and y of the type Traits::Range. When f.evaluate(x,y) is evaluated, x will be a local coordinate, and the expression should set y to the function value at that position. The initial value of y should not be used.
outVector where to store the interpolated coefficients.
