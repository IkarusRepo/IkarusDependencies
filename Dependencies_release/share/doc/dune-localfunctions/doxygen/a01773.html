Detailed Description

template<class D, class R>
class Dune::HierarchicalSimplexP2WithElementBubbleLocalBasis< D, R, 1 >

Hierarchical P2 basis in 1d.

The shape functions are associated to the following points:

f_0 ~ (0.0) // linear function f_1 ~ (1.0) // linear function f_2 ~ (0.5) // quadratic bubble
